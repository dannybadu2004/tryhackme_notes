# IP Basics

## What is IP
An address that uniquely identifies a device in a network.

---

## IP Types
IPv4 / IPv6  
Public / Private  
Static / Dynamic  

---

## Private Ranges
192.168.x.x  
10.x.x.x  
172.16–31.x.x  

---

## Commands
ip addr  
ifconfig  
ping  
traceroute  

---

## Notes
Without IP, devices can’t communicate.
