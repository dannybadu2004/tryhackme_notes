# OSI Model

## Layers
1. Physical  
2. Data Link  
3. Network  
4. Transport  
5. Session  
6. Presentation  
7. Application  

---

## Purpose
Shows how data travels through a network.

---

## Examples
Application – HTTP  
Transport – TCP  
Network – IP  
Data Link – MAC  
Physical – Cable  

---

## Notes
OSI helps troubleshoot network problems.
